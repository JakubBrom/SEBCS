Documentation of SEBCS_SA¶

SEBCS_SA includes some usable functions:-)

class scripts.SEBCS_SA.SEBCS_SA(band_red, band_nir, band_tir, DMT, ta, Rs_in=None, Rh=None, latitude=None, longitude=None, date_acq=None, timeGMT=None, output_list=None, output_folder='/home', out_file_name=None, method='aero', sat_type='L5', emis_rule='No', output_driver='GTiff', multiband=False, band_blue=None, band_green=None, band_sw1=None, band_sw2=None, U=None, h_min=None, h_max=None, mask=None, albedo=None, canopy=None, Z_st=10.0, Z=200, cp=1012)[source]¶

SEBCS_SA is class for calculation surface energy balance and crop water stress features.

Required parameters: Layers

Parameters:
  • band_red (str) – Path to raster file in red region of spectra. Spectral reflectance (rel.; 0-1).
  • band_nir (str) – Path to raster file in NIR region of spectra. Spectral reflectance (rel.; 0-1).
  • band_tir (str) – Path to surface temperature layer (raster file). Temperature (grades of Celsius).
  • DMT (str) – Path to DMT layer (raster file). Metres.
  • ta (str) – Path to air temperature layer (raster file). Temperature (grades of Celsius).

Required parameters: Constants

Parameters:
  • Rs_in (float) – Value of actual incomming global radiation flux (W.m-2) measured by pyranometer.
  • Rh (float) – Value of actual air relative humidity (%).
  • latitude (float) – Value of mean (center) latitude of area used for calculation (dec. grades, e.g.: 49.47575471).
  • longitude (float) – Value of mean (center) longitude of area used for calculation (dec. grades, e.g.: 15.30221431).
  • date_acq (datetime.date) – Date of data acquisition (ISO format, e.g.: ‘2016-05-09’).
  • timeGMT (datetime.time) – Time of data acquisition in GMT (ISO format, e.g.: ‘14:46:00.0000’).

Required parameters: Outputs

Parameters:
  • output_list (list) –

    List of output layers file names. Following names for the output layers should be used:

    Rs_in - Incoming shortwave radiation (W.m-2) Rs_out - Reflected shortwave radiation (W.m-2) albedo - Albedo (rel.) RL_in - Incoming longwave radiation (W.m-2) RL_emit - Reflected longwave radiation (W.m-2) Rn - Total net radiation (W.m-2) Ts - Surface temperature (Degree C) emis - Surface emissivity (rel.) LE - Latent heat flux (W.m-2) LEp - Latent heat flux - Penman pot. (W.m-2) LE_PT - Latent heat flux - Priestley-Taylor (W.m-2) H - Sensible heat flux (W.m-2) G - Ground heat flux (W.m-2) E_int - Evapotranspiration intensity (mmol.m-2.s-1) EF - Evaporative fraction (rel.) Bowen - Bowen ratio (-) omega - Decoupling coefficient (rel.) CWSI - CWSI (-) frict_U - Friction velocity (m.s-1) ra - Aerodynamic surface resistance (s.m-1) rc - Resistance for water vapour transfer (s.m-1) NDVI - NDVI (-) MSAVI - MSAVI (-) NDMI - NDMI (-) slope - Slope (Degree) aspect - Aspect (Degree)
  • output_folder (str) – Path to output folder where the outputs will be created.
  • out_file_name (str) – Name of exported multiband raster. Default is None.

Default: Setting

Parameters:
  • method (int) –

    Setting of calculation method. There are two calculation methods: - ‘aero’: aerodynamic method of calcultaion based

    on SEBAL method is used (default)
    • ’grad’: Method grad is used for calculation based
      on gradient approach.

    See Manual for more details.

  • sat_type (int) – Setting of satellite type which was used for calculation. There are three different satellite type: - ‘L5’: data from Landsat 5 TM (default); - ‘L7’: data from Landsta 7 ETM+; - ‘other’ - data from other satellite or aerial system. Satellite type setting is important for albedo calculation.
  • emis_rule (str) –

    Setting of emissivity calcultaion for correction of surface temperature layer. There are two variants: - ‘No’: No correction on emissivity (default) - ‘Yes’: Correction of the surface temperature layer

    on emissivity. In case the source data have emissivity equal to 1.
  • output_driver (str) – GDAL driver for output files format. Different formats should be used. GTiff is default.
  • multiband (bool) – Option of multiband raster creation. Default is False.

Optional: Spectral layers

Parameters:
  • band_blue (str) – Path to raster file in blue region of spectra. Spectral reflectance (rel.; 0-1).
  • band_green (str) – Path to raster file in green region of spectra. Spectral reflectance (rel.; 0-1).
  • band_sw1 (str) – Path to raster file in SWIR region of spectra (ca 1.61 micro meters). Spectral reflectance (rel.; 0-1).
  • band_sw2 (str) – Path to raster file in SWIR region of spectra (ca 2.2 micro meters). Spectral reflectance (rel.; 0-1).

Optional: Additional layers

Parameters:
  • U (str) – Path to wind speed layer (raster file). Velocity (m.s-1). This layer should be set if method aero is used.
  • h_min (str) – Path to minimal vegetation heigth layer (raster file). Heigth (m). This layer should be set if method aero is used.
  • h_max (str) – Path to maximal vegetation heigth layer (raster file). Heigth (m). This layer should be set if method aero is used.
  • mask (str) – Path to mask (raster layer) for the output. Mask should be real data type. Output layers are multiplied by mask where 0 are nodata and 1 are data. Mask is used for data export only. Default is None.
  • albedo (str) – Path to albedo layer (raster file). This option is used prior to calculation of albedo by SEBCS SA. Deafult is None.
  • canopy – Path to canopy heigth layer. This option is used prior to calculation of canopy heigth from the min and max heigth values. Default is None.

type canopy: str

Optional: Constants

Parameters:
  • Z_st (float) – Heigth of wind speed heigth measurement (m). Default is 10 m.
  • Z (float) – Heigth of mixed layer (m). Default is 200 m
  • cp (float) – Specific heat capacity of dry air (J.kg-1.K-1). Default cp = 1012 J.kg-1.K-1

Returns

Returns:Raster file or files for calculated variables. Number of outputs and also type of outputs are set by output_list variable (list of names). Output file(s) is saved into output_folder.
